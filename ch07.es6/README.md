## let / const

- undefined 記憶體準備位置
- not defined
- Hoisting: 變數和函數的宣告會在編譯階段就被放入記憶體
- var 作用域是 function scope
- let 作用域是 block
- const 為常數，不能被更改，不過物件內的屬性是可以更動的，因為物件是傳參考，而不是修改參考
